  • Texture Masks

What does it do?

This adds a texture node as the mask factor for the layer. When selected, it will switch you to texture paint mode automatically, in the same way the Paint Layers do.

What is it for?

This mask is only available for the layer and not for the individual channels. You can use it to paint masks or use specific textures as such.

Settings

  • Mapping: These are the usual mapping features you would expect from a texture
  • Texture slot: Here you can see which texture is currently selected
  • Open texture: This will open the filebrowser, where you can select a mask texture
  • Add mask: This will add a blank mask texture to the layer which you can paint on
