What does it do?

A Node Group Mask uses the output of a node group to affect the layers visibility. This means that you can have complex node setups to create a mask for your layer. You can then tweak the inputs of this node group to change the mask. These groups can also use the baked AO and Bevel maps to affect only parts of the mesh.

What is it for?

These masks can be used if you want to create quick but complex masks. This also allows you to take advantage of AO and bevel mask textures. If you want to, this also allows you to work with nodes, to create your own groups.

Settings

Below that, there are the settings for every input of the node group. Some inputs do special things, like creating collapsable panels or texture inputs. You can find more information about that here.

If you see a text like this, you now that the node group uses an AO or Bevel mask to change the look of the material. You can see how to bake them here. The textures will be selected automatically when their baked.

Bake Masks

Node Group masks can be baked to images with the bake button. This will ask you for a resolution and will then bake the mask to a texture which can then be edited. This can be good to save resources. Obviously it also means that you can't tweak the settings anymore.
