Usage

Layer Painter is integrated into an active tool, which can be accessed in Object and Texture Painting mode. This means, that if you want to use the addon, you need to have the active tool selected.

To get started do the following:

  • Select the active tool
  • Open the N-Panel under the Layer Painter or Tool tab or open the topbar
  • Press the Activate Layer Painter button

This will open the Layer Painter workspace which you need to use the addon.
If you close or remove the workspace you can use this button to switch to or add the workspace back.
