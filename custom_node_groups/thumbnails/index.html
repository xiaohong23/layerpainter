Generating the thumbnails

You don't have to use the provided file to generate your thumbnails, but they will look better integrated if you do. Just make sure that you use a square image; a resolution of 300x300px should be enough.

The thumbnails for the custom node groups can be created with the provided .blend file. Simply append your custom node group to that file. Depending on if you want to create a mask or a layer node group, apply the corresponding material to the Suzanne object in the scene.
Then connect the sockets in the node editor and render the image.

Using custom thumbnails

The thumbnails should be in a folder called "thumbnails". This folder needs to be located in the same folder you chose in the preferences.

Then make sure the name of the thumbnail images filename fits the name of your custom node group. If the group is called "Custom Mask__", the image should be called "Custom Mask__.png".

The thumbnails will be imported together with the node groups, when the addon is activated. You can also use the buttons in the topbar, to import them.

If no thumbnail is provided, a default image will be used. This is the silhouette of the Suzanne model.
