Known Issues

Issue

The PBR textures folder property won't open the filebrowser. This is a known bug which is not yet resolved.

Issue

The material turns pink. This is not directly an issue with the addon. It's more a limit of Blender and your graphics card.

If you have to many textures/ nodes in your material, Blender just can't handle it anymore. I hope that this improves over time.

If you want to know more about this, you can go here.

Solution

There is no real solution for this, as it's just something that need to improve with Blender, but you can do a few things to counter this:

  • Bake your material to a single layer (will obviously remove the issue, but also your layers)
  • Clear your file of unused textures
