Provided files

Use the newest Blender version

When you install Layer Painter, please make sure that you're using blender 2.80 which you can get here.

When you download the addon you will get a .zip file. You will need to unpack that file and will get two files:

layerpainter_v1.1.x.zip
This is the actual addon which you will have to install

custom_assets.zip
These are just some files that will help you create custom node groups. You don't need them for the installation.

Installing the addon

Remove previous versions!

If you have a previous version of Layer Painter installed, make sure to remove that version and then restart blender before installing the new version.

The addon is installed like every other addon in blender.

Go to the Add-ons tab in the Blender preferences and press Install.
Select the layerpainter_v1.1.x.zip and select Install Add-on from file.

Then click the checkbox and restart Blender.

Custom assets

The custom_assets.zip includes a few files, that will come in handy for creating custom node groups. These files are not required in any way, they're just there to help you create your own assets.
