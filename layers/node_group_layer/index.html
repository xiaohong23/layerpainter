What does it do?

A Node Group Layer uses the outputs of a node group to affect the channels. This means that you can have complex node setups to use in a layer of your material. You can then tweak the inputs of this node group to change the look of your material. These groups can also use the baked AO and Bevel maps to affect only parts of the mesh.

What is it for?

These layers can be used if you want to create a base for a material or get a quick result. If you want to, this also allows you to work with nodes, to create your own groups.

Settings

After the general layer settings you might see the message, that unused channels have been hidden. This will appear if the node group does not have an output for every channel.

Then there is a dropdown and a thumbnail selection for selecting a group.

Below that, there are the settings for every input of the node group. Some inputs do special things, like creating collapsable panels or texture inputs. You can find more information about that here.

If you see a text like this, you now that the node group uses an AO or Bevel mask to change the look of the material. You can see how to bake them here. The textures will be selected automatically when their baked.
