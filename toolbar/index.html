Topbar Basics

The topbar holds both, settings and tools. You can bake masks and the layers to PBR textures, as well as change settings for the workspace and the material.

Special buttons

The first button can be used to toggle the texture editor. This will open a 2D view, which you can use to view your textures.

The second button will hide all unused objects, meaning all that do not have the selected material.
