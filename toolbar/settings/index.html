Working settings

Material

  • Active Material: The selected material that you're editing
  • General Height Strength: The overall height strength of your material
  • General Normal Strength: The overall normal strength of your material
  • Clearcoat: The clearcoat amount of your material
  • Clearcoat roughness: The clearcoat roughness of your material
  • Transmission: The transmission amount of your material
  • Transmission roughness: The transmission roughness of your material

Layer

  • Default channel visibility: The default visibility of the channels for every layer you add

Painting

  • Default resolution: The default resolution for the textures created for your layers
  • Symmetry (X|Y|Z): Enables symmetry for the toggled axes

Tools

  • Import internal node groups: This will import the internal node groups of the addon
  • Import custom node groups: This will import the node groups from your selected file
  • Save textures: This will save all unsaved textures that belong to the addon
  • Invert texture: This will open a popup where you can select a texture to invert
  • Disable Layer Painter: This will remove all unused node groups that have been imported by the addon as well as remove the Layer Painter workspace

General settings

Workspace

  • Enable Layer Tags: This enables the color tags on layers
  • Enable Group Tags: This enables the color tags on groups
  • Random Color Tags: This adds a random color tag to every newly added layer
  • Compact Layer Settings: This displays the layer settings in a more compact way
  • Break Visibilities: If the Emission and the Alpha channel are enabled, the visibilites in the layer settings will break in two lines
  • Show thumbnail labels: This shows the labels of the node groups in the thumbnail selection
  • LilySurfaceScrapper integration: This shows the URL button in the Fill Layer settings
  • Datamanager integration: This shows the datamanager icon in the layer operators
  • Background image: This is a selection for the thumbnail image
  • Background rotation: This is the rotation of the background image
  • Background alpha: This is the alpga of the background image
  • Show look dev 'balls': This shows the Look Dev spheres in the viewport
  • Screen Space Reflections: This will enable SSR in eevee
  • Ambient Occlusion: This will enable ambient occlusion in eevee
  • Factor: This is the factor for the ambient occlusion setting

Simplify

  • Use simplify: This enables the simplify option
  • Max subdivisions: The maximal levels of subdivisions displayed in the viewport

Properties

  • PBR textures: The filepath of the folder with your PBR texturesets, which can be imported with the thumbnail selection in the Fill Layer settings. This should contain folders with PBR textures
